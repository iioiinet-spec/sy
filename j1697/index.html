1. 문제 이해

정올 1697번 큐(queue)는 큐 자료구조를 직접 구현하는 문제입니다. 명령어가 주어질 때마다 큐에 값을 넣거나, 맨 앞 값을 꺼내거나, 현재 큐의 크기를 출력해야 합니다.

명령 뜻 예시
i a 숫자 a를 큐에 넣기 i 10
o 큐의 맨 앞 숫자를 꺼내 출력 o
c 큐에 들어 있는 데이터 개수 출력 c
큐가 비어 있는데 o 명령이 나오면 숫자를 꺼낼 수 없으므로 empty를 출력합니다.

2. 큐(Queue) 개념

FIFO First In First Out
먼저 들어온 데이터가 먼저 나갑니다.
front 큐의 맨 앞 위치입니다.
pop할 때 이 위치의 값을 꺼냅니다.
rear 새 데이터가 들어갈 위치입니다.
push할 때 이 위치에 넣습니다.

예시

큐에 10, 20, 30을 차례대로 넣으면 다음과 같습니다.

입력 순서: 10 → 20 → 30
출력 순서: 10 → 20 → 30

가장 먼저 들어온 10이 가장 먼저 나옵니다.

4. 변수 설명

int queue[MAX];
int front = 0;
int rear = 0;
변수 의미
queue 큐 데이터를 저장하는 배열
front 가장 앞 데이터의 위치
rear 새 데이터가 들어갈 위치

4.1 push 함수

void push(int x) {
    queue[rear] = x;
    rear++;
}

새로운 숫자를 rear 위치에 넣고, rear를 1 증가시킵니다.

4.2 pop 함수

int pop() {
    int value = queue[front];
    front++;
    return value;
}

큐의 맨 앞인 front 위치의 값을 꺼내고, front를 1 증가시킵니다.

4.3 size 함수

int size() {
    return rear - front;
}

큐에 남아 있는 데이터 개수는 rear - front입니다.

4.4 empty 함수

int empty() {
    return front == rear;
}

front와 rear가 같으면 큐가 비어 있다는 뜻입니다.

5. 정답 C 코드

#include <stdio.h>

#define MAX 101

int queue[MAX];
int front = 0;
int rear = 0;

void push(int x) {
    queue[rear] = x;
    rear++;
}

int pop() {
    int value = queue[front];
    front++;
    return value;
}

int size() {
    return rear - front;
}

int empty() {
    return front == rear;
}

int main() {
    int N;
    scanf("%d", &N);

    for (int i = 0; i < N; i++) {
        char command;
        scanf(" %c", &command);

        if (command == 'i') {
            int x;
            scanf("%d", &x);
            push(x);
        }
        else if (command == 'o') {
            if (empty()) {
                printf("empty\n");
            } else {
                printf("%d\n", pop());
            }
        }
        else if (command == 'c') {
            printf("%d\n", size());
        }
    }

    return 0;
}
scanf(" %c", &command); 에서 %c 앞의 공백은 매우 중요합니다. 줄바꿈 문자나 공백을 건너뛰고 실제 명령 문자를 읽게 해 줍니다.

7. 시간복잡도

명령 작업 시간복잡도
irear 위치에 값 넣기O(1)
ofront 위치의 값 꺼내기O(1)
crear - front 계산O(1)
명령이 N개라면 각 명령은 O(1)이므로 전체 시간복잡도는 O(N)입니다.

8. 핵심 정리

  1. 큐는 먼저 들어온 데이터가 먼저 나가는 FIFO 자료구조입니다.
  2. front는 꺼낼 위치입니다.
  3. rear는 새 값이 들어갈 위치입니다.
  4. push는 rear 위치에 값을 넣고 rear를 증가시킵니다.
  5. pop은 front 위치의 값을 꺼내고 front를 증가시킵니다.
  6. 큐의 크기는 rear - front입니다.
  7. front와 rear가 같으면 큐가 비어 있습니다.
  8. 각 명령은 O(1)에 처리됩니다.
  9. 전체 시간복잡도는 O(N)입니다.
